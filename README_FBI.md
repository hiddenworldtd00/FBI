Un terminal immersif de style FBI qui transforme votre environnement Linux en centre d'opérations cybernétiques. Inspiré des films et séries d'espionnage, ce script crée une expérience utilisateur unique avec multiples fenêtres d'outils, surveillance en temps réel, et interface futuriste.

## 🎯 Objectif

Créer un environnement de travail immersif pour :
- La cybersécurité défensive et offensive
- L'analyse forensique
- La surveillance réseau
- L'OSINT (Open Source Intelligence)
- L'apprentissage des outils de sécurité

## 🚀 Installation

```bash
# Cloner le repository
git clone https://github.com/hiddenworld/FBI_CyberOps.git
cd FBI_CyberOps

# Installer les dépendances
pip install -r requirements_FBI.txt

# Lancer le terminal FBI
python FBI_CyberOps.py
```

## 🎮 Utilisation

### Lancement Immersif
```bash
python FBI_CyberOps.py
```

Le script va :
1. Transformer le terminal avec un thème FBI
2. Afficher le logo animé du FBI
3. Ouvrir plusieurs fenêtres d'outils en parallèle
4. Démarrer la surveillance système en temps réel

### Modes d'Opération

#### 1. Mode Surveillance (S)
- Surveillance réseau en temps réel
- Détection d'intrusions
- Analyse de paquets
- Monitoring système

#### 2. Mode Investigation (I)
- OSINT et reconnaissance
- Analyse de vulnérabilités
- Forensics numérique
- Recherche d'informations

#### 3. Mode Opération (O)
- Outils de pentest
- Exploitation (simulation)
- Post-exploitation
- Maintien d'accès

#### 4. Mode Analyse (A)
- Analyse de malware
- Reverse engineering
- Cryptographie
- Stéganographie

#### 5. Mode Terminal (T)
- Terminal personnalisé FBI
- Historique des commandes
- Scripts automatisés
- Configuration système

#### 6. Mode Rapport (R)
- Génération de rapports
- Export des données
- Visualisations
- Timeline des événements

## 🖥️ Transformation du Système

Le script modifie temporairement :
- Le prompt du terminal (style FBI)
- Les couleurs et le thème
- Le titre de la fenêtre
- L'historique des commandes
- Les alias de commandes

Toutes les modifications sont réversibles en quittant le script.

## 📊 Outils Intégrés

### Surveillance
- `nmap` - Scan de ports
- `tcpdump` - Capture de paquets
- `wireshark` - Analyse réseau
- `netstat` - Connexions réseau
- `iftop` - Bande passante

### Investigation
- `theHarvester` - OSINT emails
- `maltego` - Analyse de liens
- `shodan` - Recherche d'appareils
- `whois` - Informations domaines
- `dig` - DNS lookup

### Forensics
- `autopsy` - Analyse forensique
- `sleuthkit` - Outils forensiques
- `volatility` - Analyse mémoire
- `binwalk` - Analyse binaire
- `strings` - Extraction de texte

### Pentest
- `metasploit` - Framework d'exploitation
- `burpsuite` - Proxy web
- `sqlmap` - Injection SQL
- `nikto` - Scan web
- `gobuster` - Brute force directories

## 🔧 Configuration

### Fichier de Configuration
```json

  "theme": "fbi_dark",

  "log_level": "INFO",
  "tools_path": "/usr/share/tools",

  "network_interface": "eth0",
  "monitoring_enabled": true
}
```
